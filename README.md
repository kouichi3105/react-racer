# React Racer

## 概要
jetracerをスマホから操作するためのもの

(Reactを勉強するために製作)

## 用意するもの

・jetracer (https://github.com/NVIDIA-AI-IOT/jetracer)

## 使い方

```
git clone https://github.com/kouichi3105/react-racer

cd react-racer

cd react && npm install && npm run build

pip install flask flask_cors

python ../flask/run.py

```
